## Thinking Frame
This exact problem — or something structurally identical — has already been solved somewhere. Biology solves resource allocation. Logistics solves flow optimization. Music composition solves constraint-based generation. Urban planning solves decentralized coordination. Find the analogy and import the proven solution.

## Approach
1. Abstract the problem to its structural essence — strip the domain-specific vocabulary
2. Find 2-3 completely different fields that deal with the same structural challenge
3. Describe how those fields solved it (specifically — not vaguely "like a river")
4. Translate that solution back into the original domain, preserving the core mechanism

## When Most Effective
Novel problems, performance optimization, distributed systems, consensus problems, anything involving flow, queues, resource contention, or coordination.

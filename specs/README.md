# Specs

This folder contains the design specifications that were used to create the Creative Loop skill.

These specs are **not part of the skill itself** — they are the planning and design documents that informed its implementation. The actual skill lives in `SKILL.md` at the project root.

## Contents

- `overview.md` — High-level system overview
- `architecture.md` — Component architecture
- `implementation-plan.md` — Implementation roadmap
- `generator.md` — Idea generation phase design
- `evaluator.md` — Evaluation phase design
- `experiment-runner.md` — Experiment runner design
- `meta-controller.md` — Meta-controller design
- `memory.md` — Memory system design
- `self-optimization.md` — Self-optimization design
- `integration.md` — Integration design
- `human-oversight.md` — Human oversight design
